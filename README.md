### :scroll: Description 

Anime Wolf is an application to explore anime, its detailes, share it with a friend and track anime episodes you watch.

### :bulb: Libraries Used  

-Retrofit: for Rest API

-Firebase Authentication

-Firebase Firestore Database

-Coil: to load Images

-Shared Preferences

-MVVM Pattern

-Data Binding

-Work Manager
